This file is a template and never ships (`draft: true` excludes it from builds).

To publish a post:

1. Copy this file to `src/content/writing/<slug>.mdx` — the filename becomes the URL.
2. Fill in `title`, `date`, `summary`, and `tags`.
3. Set `draft: false`.

The post appears automatically on `/writing/`, in the home page Recent Writing
section, in `rss.xml`, and in the sitemap. Flip `showWriting` in
`src/data/site.ts` to add Writing to the navbar.
